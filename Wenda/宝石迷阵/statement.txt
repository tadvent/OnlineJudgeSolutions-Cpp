题目描述

完美世界的手机团队最近收到一个需求，需要制作一个手机小游戏，宝石迷阵。这款游戏一共有4种宝石分为A B C D。当这4种宝石在横排竖排中有相同的连续多个(>=3)的时候就可以自动消除。消除完成以后上面一行的宝石会自动填充被消除的宝石。但是不会有新的宝石掉落。

A B C D 4种宝石可以同时消除

宝石一次消除以后会自行下落，下落以后又可能组成新的消除，依次循环！


输入格式

一共有A B C D 4种宝石分别用‘A’ , ’B’, ‘C’ ,’D’ 字符表现

第一行输入1个整数 C 表示一共要输入的矩阵数量

下面2行各输入1个整数  M，N （4<=M,N <= 10）分别代表矩阵的行数和列数。

下面的M行输入每行N个非空字符（A B C D）。中间无间隔


输出格式

输出每个迷阵一次迭代的状态，消除空白的地方写‘0’

测试格式

输出每次消除的矩阵状态
